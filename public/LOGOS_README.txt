INSTRUÇÕES PARA ADICIONAR AS LOGOS DO TENTÁCULO FLOW
=====================================================

Por favor, adicione os seguintes arquivos nesta pasta (public/):

1. logo.png - Logo principal do Tentáculo Flow
2. logo-auth.png - Logo para a página de autenticação
3. bg-auth.png - Imagem de fundo para a página de autenticação
4. favicon.png - Favicon do site

Após adicionar as logos, o sistema estará completo e pronto para uso.

OBSERVAÇÕES:
- As logos devem estar em formato PNG com fundo transparente
- Tamanhos recomendados:
  * logo.png: 200x200px ou maior
  * logo-auth.png: 200x200px ou maior
  * bg-auth.png: 1920x1080px ou maior
  * favicon.png: 32x32px ou 64x64px
